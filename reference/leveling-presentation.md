# Leveling Presentation

How and when level-ups get offered and presented. Established 2026-07-22.

## Cadence
Level-ups are offered roughly at the end of each arc — not every arc guaranteed, but that's the default beat to check against. LL1 was offered at the close of Arc 1 (Vantage Tariff Racket) and felt well-timed. Arcs 3 and 4 closed without a level-up being offered — the next arc's close (or sooner, if the GM's judgment says an arc's stakes earned it early) is where the next one is due.

## Presentation
Lead with a **short summary** — available options at the new level, name + one-line effect, not full text. Pull the **full text** from `talents-reference.md` / `triggers-reference.md` only for options actually being considered, not the whole list up front. (LL1 used a full dump of both references before any choice was made — that was fine for a first level-up with no prior context, but isn't the default going forward.)

## Choice format
Present the level's individual choices (mech skill point, license rank, talent rank, trigger point, core bonus, reallocation, etc.) as a structured multi-question choice set, one question per choice, each option carrying a short mechanical explanation — the same format used when codifying these GM systems (loot/interrogation/downtime/reward/leveling itself). Submit all the choices for that level-up together in one batch rather than resolving them one at a time in freeform back-and-forth.
